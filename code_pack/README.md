# 현대오토에버 14일 Java 코드 팩

- 각 Day 폴더에는 `master/Solution.java`, `practice_blank/Solution.java`, `practice_answer/Solution.java`가 있습니다.
- 모든 코드는 프로그래머스 제출 형식의 `class Solution`입니다.
- 폴더별로 따로 열어 사용하세요. 같은 폴더에 여러 `Solution` 클래스를 두지 마세요.
- 권장 순서: master 입력 -> practice_blank 복원 -> practice_answer 비교.

- Day 1: 구현·시뮬레이션: 상태를 그대로 옮기기
- Day 2: BFS: 같은 비용의 최단 이동
- Day 3: DFS: 연결된 덩어리 찾기
- Day 4: 정렬·그리디: 지금의 선택 기준 만들기
- Day 5: 우선순위 큐: 가장 급한 후보부터 꺼내기
- Day 6: 다익스트라: 비용이 다른 최단경로
- Day 7: 1주차 복습: 유형을 먼저 맞히는 미니 모의
- Day 8: 기본 DP: 이전 답으로 현재 답 만들기
- Day 9: 배낭 DP: 한정된 자원으로 최대 가치
- Day 10: 백트래킹: 선택하고 되돌리기
- Day 11: 누적합·이분 탐색: 빠르게 합치고 답을 좁히기
- Day 12: 비트마스크 BFS: 위치와 수집 상태를 함께 저장
- Day 13: 다익스트라 전처리 + 비트마스크 DP
- Day 14: 최종 모의고사: 그리디 + 최단경로·비트마스크 DP
